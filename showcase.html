א heb-date-picker
Angular · TS · RTL

Hebrew calendar date picker

לוח עברי לבחירת תאריכים, בגימטריה.

רכיב Date Picker ל-Angular שמציג את הלוח העברי עם תאריכים כתובים כהלכה — כ״א אלול תשפ״ו. טווח, מגבלות, סימון חגים ושבתות, RTL מלא, ואינטגרציה עם טפסים. כל חישובי הלוח נשענים על @hebcal/core.

$ npm i heb-date-picker תיעוד ה-API

Capabilities

כל מה שצריך מבורר תאריכים

ארבע יכולות הליבה שביקשתם — ועוד. כל תא בלוח מגיע עם דגלים מחושבים מראש, וה-UI רק מצייר אותם.

📅

יום יחיד או טווח

mode="single" או mode="range" — בחירת תאריך בודד או טווח עם הדגשת הימים שביניהם.

🚫

מגבלות וימים חסומים

min, max, ופרדיקט disabledDates להשבתת ימים לפי כל כלל.

🕯️

חגים ושבתות

סימון אוטומטי של חגים, מועדים, שבתות וראשי-חודשים — ללוח א״י או לתפוצות (israel).

🔤

ניקוד לפי בחירה

showNikud מחליף בין אלול ל-אֱלוּל בכותרות ובטקסט.

🧾

Angular Forms

מממש ControlValueAccessor — עובד עם [(ngModel)] ו-Reactive Forms.

🎨

מיתוג ב-CSS

Quick start

שלושה צעדים לשימוש

app.component.ts 
// 1. npm i heb-date-picker  →  2. import  →  3. use
import { Component, signal } from '@angular/core';
import { FormsModule } from '@angular/forms';
import { HebDatePickerComponent, PickerValue } from 'heb-date-picker';

@Component({
  selector: 'app-root',
  standalone: true,
  imports: [HebDatePickerComponent, FormsModule],
  template: `
    <heb-date-picker [(ngModel)]="date" [showNikud]="true">
    </heb-date-picker>
  `,
})
export class App {
  date = signal<PickerValue>(null); // Date | null
}
