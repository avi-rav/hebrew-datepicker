Hebrew calendar date picker · Angular

לוח עברי לבחירת תאריכים, בגימטריה.

רכיב Date Picker ל-Angular שמציג את הלוח העברי עם תאריכים כתובים כהלכה — כ״א אלול תשפ״ו. נסו את הלוח משמאל: החיצים באמת מנווטים בין חודשים ושנים.

$ npm i heb-date-picker כל ההדגמות ↓
כ״א אֱלוּל תשפ״ו
הפורמט מגיע מ-renderGematriya(): יום, חודש ושנה באותיות עבריות, עם ניקוד או בלעדיו — כולל ט״ו / ט״ז ושנים מעוברות (אדר א׳ / אדר ב׳). הלוח כאן מחשב הכל בזמן אמת.
